# Lane 2 — PR Source Diversification (don't rely on Qwoted alone)

Goal: stop single-threading digital PR on Qwoted. Same expert, same pitch bank, more
shots on goal. Qwoted stays automated (weekly task); these add resilience.

> **Reality check:** only Qwoted has a usable API (the automated weekly task). The others
> below are **email-digest / web-app based** — they can't be auto-sent the same way, but
> with the ready pitch bank each response takes ~5 minutes. Treat this as a 15-min/week
> manual habit layered on top of the automation.

## Workflow (same every time)
1. Subscribe to each with the profile above; pick **Health/Beauty + Small Business** categories.
2. Triage incoming requests against your real expertise (skip anything needing a derm/MD).
3. Draft fast from `haro-pitch-bank.md` — swap sentence one to mirror the reporter's exact
   wording. Keep ~150 words, no medical claims, founder credit, link only if allowed.
4. Send from support@eliraliving.com before the deadline.
5. Log sent responses in `haro-sources.md` tracking table (so you don't double-pitch).

## Rules (same as Qwoted)
- Answer the exact question in sentence one. No preamble, no PR fluff.
- Plain, appearance-based claims only — never "treats/cures/heals".
- Disclose you're a founder (it's your credibility, and it's honest).
- One quality placement on a real publication beats 50 directory links.

## Possible future automation
Featured.com and SourceBottle have no open pitch API, so they stay manual for now. If
either ships an API (or via their email → an n8n parser), we can fold them into the weekly
task the way Qwoted is. For now: Qwoted = automated, these = 15 min/week by hand.
